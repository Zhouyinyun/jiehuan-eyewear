# 界环眼镜门店查询系统

这是一个简单的门店查询系统，帮助用户快速找到距离自己最近的界环眼镜门店。

## 功能特点

- 自动定位用户当前位置
- 展示附近的界环眼镜门店
- 显示门店详细信息和距离
- 支持地图查看和导航
- 支持手动选择城市

## 技术栈

- HTML5
- CSS3
- JavaScript
- 高德地图 API

## 项目结构

```
├── index.html          # 主页面
├── css/               # 样式文件
│   └── style.css      
├── js/                # JavaScript文件
│   ├── main.js        # 主要逻辑
│   ├── map.js         # 地图相关
│   └── data.js        # 数据处理
├── data/              # 数据文件
│   └── stores.json    # 门店数据
└── README.md          # 项目说明
```

## 使用说明

1. 直接打开index.html即可使用
2. 允许浏览器获取位置信息
3. 系统会自动显示离你最近的门店

## 注意事项

- 需要允许浏览器定位权限
- 需要网络连接以加载地图
